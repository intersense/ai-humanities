# AI-Humanities

## 介绍
本项目展示数字人文研究中广泛应用的文本挖掘、网络分析、自然语言处理相关技术的应用代码案例。

## 安装依赖包
生成依赖包文件：
```python
pip list --format=freeze > requirements.txt
```

```python
pip install -r requirements.txt
``` 

在Jupyter中将需要的包安装到当前的kernel使用如下命令：
```python
import sys
!{sys.executable} -m pip install wordcloud
```
